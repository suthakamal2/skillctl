## When this applies
Loads when the prompt mentions: i18n, localization, translation, locale.

## Guidance
- Externalise strings, format dates/numbers per locale, no string concatenation.
- Make the safe choice the default; document the exception when you deviate.
- Prefer the boring, well-understood option over the clever one.
