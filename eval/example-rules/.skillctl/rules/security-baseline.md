# Security Baseline

Never log secrets; validate all input; least privilege by default.

## When this applies
Always-on behavioural policy.

## Guidance
- Never log secrets; validate all input; least privilege by default.
- Make the safe choice the default; document the exception when you deviate.
- Prefer the boring, well-understood option over the clever one.
