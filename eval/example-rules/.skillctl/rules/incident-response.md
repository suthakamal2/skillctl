# Incident Response

Declare early, assign a lead, communicate, mitigate first, blameless postmortem.

## When this applies
Loads when the prompt mentions: incident, outage, sev1, postmortem.

## Guidance
- Declare early, assign a lead, communicate, mitigate first, blameless postmortem.
- Make the safe choice the default; document the exception when you deviate.
- Prefer the boring, well-understood option over the clever one.
