# Frontend State

Keep server state in a query cache; minimal global client state.

## When this applies
Loads when the prompt mentions: state management, redux, store, client state.

## Guidance
- Keep server state in a query cache; minimal global client state.
- Make the safe choice the default; document the exception when you deviate.
- Prefer the boring, well-understood option over the clever one.
