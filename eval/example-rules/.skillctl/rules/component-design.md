# Component Design

Small, composable, single-responsibility components; lift state deliberately.

## When this applies
Loads when the prompt mentions: component, react, ui component, props.

## Guidance
- Small, composable, single-responsibility components; lift state deliberately.
- Make the safe choice the default; document the exception when you deviate.
- Prefer the boring, well-understood option over the clever one.
